  • 관심기술

    필수 기술

    HTML/CSS: 웹 페이지의 구조와 스타일 정의
    JavaScript: 웹 페이지 동작 구현에 사용

    프레임워크 및 라이브러리

    React: 사용자 인터페이스를 구축
    Angular: 복잡한 웹 애플리케이션을 구축하는 데 유용한 프레임워크
    Vue.js: 점진적으로 적용 가능한 UI 프레임워크

    스타일링 도구

    Sass/SCSS: CSS 전처리기로, 더 복잡하고 유지 보수하기 쉬운 스타일 작성에 사용
    Tailwind CSS: 유틸리티 우선 CSS 프레임워크

    상태 관리

    Redux: 애플리케이션의 상태를 중앙에서 관리하기 위한 도구
    Context API (React): React 애플리케이션에서 전역 상태를 관리하는 데 사용

    빌드 도구

    Webpack: 모듈 번들러로, 여러 파일을 하나의 파일로 번들링하는 데 사용
    Parcel: 설정이 적고 사용하기 쉬운 빌드 도구

    테스트

    Jest: JavaScript 테스팅 프레임워크로, 주로 React 애플리케이션에서 사용
    Cypress: 엔드 투 엔드 테스트 프레임워크로, 전체 애플리케이션을 테스트

    클라우드 및 배포

    Netlify/Vercel: 정적 사이트를 쉽게 배포하고 관리할 수 있는 플랫폼
    Netlify/Vercel: 정적 사이트를 쉽게 배포하고 관리할 수 있는 플랫폼

    기타

    TypeScript: JavaScript에 정적 타입을 추가하여 코드의 안정성과 가독성을 높임
    GraphQL: 데이터를 요청하는 데 사용되는 쿼리 언어로, API와의 상호작용을 더 효율적으로 만듦
    Responsive Web Design: 다양한 장치에서 잘 보이도록 웹 페이지를 디자인하는 기술

  • 더 알아보고 싶은 기술들

    React 18:

    성능 향상 및 사용자 경험 개선을 목표로 한 여러 기능 도입

    Server Components:

    서버에서 컴포넌트를 렌더링하여 클라이언트 측에서의 초기 로드 시간을 줄인다.

    Next.js 13:

    React 기반의 프레임워크. 새로운 App Router 및 Turbopack 빌드 도구를 도입.

    Vue 3:

    Composition API, 더 나은 TypeScript 지원, 성능 향상 등

    SvelteKit:

    Svelte를 기반으로 한 애플리케이션 프레임워크.

    Vite:

    빠른 개발 서버와 최적화된 빌드를 제공하는 차세대 프론트엔드 빌드 도구

    Web Components:

    다른 프레임워크와 상호 운용 가능한 독립적인 UI 구성 요소를 만들 수 있게 해준다.

    Micro-Frontends:

    대규모 애플리케이션을 작은 독립적인 단위로 나누어 관리하는 아키텍처

    CSS-in-JS:

    avaScript 내에서 스타일을 관리하여 컴포넌트 기반 개발을 더 쉽게 한다.

    WebAssembly (Wasm):

    브라우저 내에서 고성능 애플리케이션을 실행할 수 있게 해주는 이진 포맷

    Edge Computing:

    클라우드 대신 사용자와 가까운 곳에서 서버를 운영

    Jamstack:

    JavaScript, API, 그리고 Markup을 활용하여 빠르고 안전한 웹사이트 구축
